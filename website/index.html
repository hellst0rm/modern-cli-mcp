Quick Start

Add to your Claude Desktop config:

{
  "mcpServers": {
    "modern-cli": {
      "command": "nix",
      "args": ["run", "github:hellst0rm/modern-cli-mcp", "--"]
    }
  }
}

Installation Options

Nix (Flakes)

nix run github:hellst0rm/modern-cli-mcp

Requires flakes enabled

Nix (Classic)

nix-build ...archive/main.tar.gz

No flakes required

Docker

docker run ghcr.io/hellst0rm/modern-cli-mcp

Container runtime

FlakeHub

fh add hellst0rm/modern-cli-mcp
